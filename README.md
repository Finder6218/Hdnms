# 单位分房管理系统

系统包含功能：
1、基本信息：住房基本信息包含选房信息
2、补交款管理：补交登记
3、信息查询：欠款查询、选择等级查询、小区查询、职称查询、是否选房查询、已交款查询、补交费用查询
# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
